# Web Design Assignment: Employee Salary Calculator

A simple, responsive web application built using HTML, CSS, and JavaScript that calculates and displays the net salaries of employees in a clean tabular format.

## 🚀 Features
* **Dynamic Calculations:** Computes net salary using JavaScript logic ($Net = Gross - Deductions$).
* **Clean UI:** Styled using modern, responsive CSS for a professional look.
* **Formatted Output:** Displays results dynamically inside an HTML table with proper currency formatting (`RWF`).
* **Structured Data:** Manages an array dataset of exactly 10 comprehensive employee records.

---

## 🛠️ Technologies Used
* **HTML5:** To structure the page layout and data table.
* **CSS3:** For modern typography, spacing, styling, and hover effects.
* **JavaScript (ES6+):** For data storage, array iteration (`forEach`), dynamic DOM manipulation, and calculation logic.
